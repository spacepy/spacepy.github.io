spacepy.pybats.pwom¶

PyBats submodule for handling input/output for the Polar Wind Outflow Model (PWOM), one of the choices for the PW module in the SWMF.

Classes

Efile(filename[, starttime])

Class for loading and parsing ionospheric electrodynamics ascii files.

Line(filename[, starttime])

Class for loading a single field line output file.

Lines(lines[, starttime])

A class for reading and plotting a complete set of PWOM line output files.
