spacepy.pybats.bats.MagGridFile¶

class spacepy.pybats.bats.MagGridFile(*args, **kwargs)[source]¶

Magnetometer grids are a recent addition to BATS-R-US: instead of specifying a small set of individual stations, the user can specify a grid of many stations spanning a latitude/longitude range. The files are output in the usual spacepy.pybats.IdlFile format. This class handles the reading, manipulating, and visualization of these files.
